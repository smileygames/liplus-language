# Li+ Wiki

Li+（liPlus）は、**構造駆動型AI開発のための言語・プロトコル**です。

AIが安全に間違えられる最小構造を定義し、実行結果を観測し、証拠にもとづいて反復する——推測と事実を混同しないために。

Li+は新しい構文を定義しません。**AIがどう判断し、どう実行し、どこで止まるかを定義します。**
